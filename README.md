### 棋譜のパース ###

"7六歩" の場合 "7六" と "歩" に分離する。
"2二角成" の場合 "2二" と "角" と "成" に分離する。
同銀の場合、同がどこを差しているのか、前の座標を見る。
"5八金右" の場合、5八から見て右下にある金が斜め上に上がったという意味なのでこの解釈が難しい。
"4八" に金があった場合、"5八金右" は真横の金なのか、斜め下の金なのか、どっちだろう。
